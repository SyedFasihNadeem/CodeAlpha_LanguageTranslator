# CodeAlpha Language Translation Tool

Task 1 of CodeAlpha Artificial Intelligence Internship.

## Live Demo
https://v0-internship-task-completion-swart.vercel.app/translator

## Project Description
A web-based language translation tool that translates text between 30+ languages in real-time using Google Gemini 2.5 Flash through the Vercel AI SDK.

## Features
- 30+ languages including English, Spanish, French, German, Urdu, Arabic, Chinese, Japanese
- Auto language detection
- Text-to-speech output
- One-click copy to clipboard
- Instant language swap
- Clean, responsive UI

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Vercel AI SDK
- Google Gemini 2.5 Flash
